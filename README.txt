DUY HEALTH HOMEPAGE - PROFESSIONAL VERSION

This package upgrades your homepage to a more attractive and professional layout with images.

FILES INCLUDED
- index.html
- styles.css
- offers.js
- images/hydrolean-xt.jpg
- images/cholibrium.jpg
- images/prodentim.jpg

HOW TO USE
1. Open your GitHub repo folder.
2. Copy all files and the images folder from this package into the root of your repo.
3. Replace old files if asked.
4. Commit and push.
5. Vercel will redeploy automatically.

HOW TO ADD NEW OFFERS LATER
Open offers.js and add another object like this:

{
  title: "Offer Name",
  category: "Offer Category",
  description: "Short description",
  url: "/your-offer-folder",
  image: "images/your-image.jpg"
}

IMPORTANT
- Make sure each URL really exists in your site.
- Make sure each image path exists in the images folder.
